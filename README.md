# how to

- download kode dalam zip
- ekstrak zip nya
- pergi ke `chrome://extensions/` dan klik "Load unpacked"
- pilih folder hasil ekstrak tadi untuk di import sebagai ekstensi (pastikan folder yang dalemnya ada `manifest.json`)
- enjoy
